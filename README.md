# Terra-Unveiled

**Terra-Unveiled** は、現実世界にゲームの「戦場の霧（Fog of War）」の概念を持ち込んだ、探索型ライフログ・マップアプリです。

## コンセプト

世界は最初、深い霧（暗闇）に包まれています。
あなたが実際にその場所を訪れる（GPSログを取得する）ことで、地図上の霧が晴れ、世界が少しずつ姿を現します。

「世界中のどこに行ったか」ではなく、**「自分の足でどこを歩き、どれだけ世界を解き明かしたか」**を可視化する、究極の探索体験を提供します。

## 主な機能

- **戦場の霧（Fog of War）オーバーレイ**: 地図全体を覆う半透明の霧。未踏の地は暗闇に包まれています。
- **リアルタイムGPS追跡**: 移動に合わせて、現在地周辺の霧がリアルタイムで晴れていきます。
- **探索シミュレーション**: 地図上をクリックすることで、その場所を「訪れた」ことにして霧を晴らすことができます（開発・テスト用）。
- **データの永続化**: 訪れた場所のデータはブラウザ（localStorage）に保存されるため、アプリを閉じても探索の進捗は失われません。
- **重複防止ロジック**: 短距離の移動ではデータポイントを追加しない最適化により、パフォーマンスを維持します。

## 技術スタック

- **Frontend**: React (TypeScript)
- **Build Tool**: Vite
- **Map Library**: Leaflet / React-Leaflet
- **Graphics**: HTML5 Canvas (霧の描画と透過処理)
- **Storage**: localStorage (探索データの保存)

## セットアップと実行方法

### 1. 依存関係のインストール

```bash
npm install
```

### 2. 開発サーバーの起動

```bash
npm run dev
```

### 3. モバイル端末でのテスト（推奨）

実際に外を歩いて試すには、PCと同じWi-Fiに接続したスマホからアクセスしてください。

```bash
npm run dev -- --host
```

表示されたIPアドレス（例: `http://192.168.x.x:5173`）にスマホのブラウザでアクセスし、位置情報の利用を許可してください。

## 今後の展望（ロードマップ）

- **階層的な霧**: 通った回数が多い場所ほど霧が完全に晴れ、明るく光るエフェクトの追加。
- **探索統計**: 自分が世界の何パーセントを解き明かしたかを算出する機能。
- **IndexedDBへの移行**: より大量の移動ログを高速に処理するためのデータベース拡張。
- **チーム探索**: 複数のユーザーで協力して特定のエリアの霧を晴らすイベント機能。
