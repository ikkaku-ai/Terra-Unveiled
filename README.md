# Terra-Unveiled

**Terra-Unveiled** は、現実世界にゲームの「戦場の霧（Fog of War）」の概念を持ち込んだ、探索型ライフログ・マップアプリです。

## コンセプト

世界は最初、深い霧（暗闇）に包まれています。
あなたが実際にその場所を訪れる（GPSログを取得する）ことで、地図上の霧が晴れ、世界が少しずつ姿を現します。

「世界中のどこに行ったか」ではなく、**「自分の足でどこを歩き、どれだけ世界を解き明かしたか」**を可視化する、究極の探索体験を提供します。

## プロジェクト構成

```text
src/
├── components/
│   ├── map/
│   │   ├── WorldMap.tsx    # 地図本体と探索ロジックのメインコンポーネント
│   │   └── FogOverlay.tsx  # Canvasを使用して地図上に霧を描画するレイヤー
│   └── ui/
│       └── WelcomeModal.tsx # 初回起動時のチュートリアル・注意喚起モーダル
├── styles/
│   ├── global.css          # 全体のリセットスタイルと基本設定
│   └── App.css             # アプリケーション固有のスタイル
├── App.tsx                 # メインのエントリーコンポーネント
└── main.tsx                # Reactのレンダリング開始点
```

### ファイルの役割

- **WorldMap.tsx**: 
  Leafletを使用した地図表示、GPSによる現在地追跡、訪問ポイントの管理（保存・読み込み）、探索度スコアの計算など、アプリの核心となる機能が詰まっています。
- **FogOverlay.tsx**: 
  HTML5 Canvasを活用し、地図の動きに同期して「未踏の地の暗闇」と「訪れた場所の穴あき（霧消）」を高速に描画します。
- **WelcomeModal.tsx**: 
  新規ユーザーへの歓迎メッセージと、安全に使用するためのガイドラインを表示します。一度閉じるとブラウザに記憶され、次回からは表示されません。
- **global.css**: 
  ブラウザ間の表示の差異をなくし、地図が画面いっぱいに広がるようにレイアウトを調整します。

## 主な機能

- **戦場の霧（Fog of War）オーバーレイ**: 地図全体を覆う半透明の霧。未踏の地は暗闇に包まれています。
- **リアルタイムGPS追跡**: 移動に合わせて、現在地周辺の霧がリアルタイムで晴れていきます。
- **探索統計**: 訪問ポイント数に基づき、擬似的な「探索度（%）」を表示します。
- **データの永続化**: 探索データはブラウザ（localStorage）に自動保存されます。外部サーバーには一切送信されません。

## セットアップと実行方法

### 1. 依存関係のインストール

```bash
npm install
```

### 2. 開発サーバーの起動

```bash
npm run dev
```

### 3. モバイル端末でのテスト（推奨）

```bash
npm run dev -- --host
```

## セキュリティとプライバシー

- **完全ローカル管理**: あなたの行動履歴（緯度・経度）はすべてブラウザのローカルストレージにのみ保存されます。ネットワークを介して外部に送信されることはありません。
- **オープンデータ**: 地図データには OpenStreetMap を使用しており、APIキーの入力や追加費用は不要です。
