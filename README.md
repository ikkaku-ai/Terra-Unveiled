# Terra-Unveiled

**「世界を、自分の足で解き明かせ。」**

Terra-Unveiled は、現実世界にゲームの「戦場の霧（Fog of War）」の概念を持ち込んだ、新感覚の探索型マップアプリです。

## 🚀 アプリを体験する

以下のリンクから、今すぐブラウザで探索を始めることができます。  
**[https://terra-unveiled.vercel.app/](https://terra-unveiled.vercel.app/)**  
*(※位置情報の利用を許可してご利用ください。)*

## 🌍 コンセプト

世界は最初、深い霧（暗闇）に包まれています。  
地図を検索して目的地へ行くだけの日常を、冒険に変えましょう。  
あなたが実際に歩いた場所だけが、地図上の霧を晴らし、鮮やかな世界を姿を現します。

「どこへ行ったか」ではなく、**「どれだけ世界を解き明かしたか」**を記録する、究極のライフログ体験を提供します。

## ✨ 主な機能

- **戦場の霧（Fog of War）**: 未踏の地は暗闇に。一歩進むごとに世界が広がるワクワク感を体験。
- **レベル・ランクシステム**: 探索率（%）に応じて称号がアップ。新米から「伝説の探検家」を目指しましょう。
- **ダークモード対応**: 夜の探索にも最適な、スタイリッシュな夜間モードを搭載。
- **リアルタイムGPS追跡**: あなたの歩みに合わせて、リアルタイムに地図が晴れていきます。
- **オートセーブ**: 探索データは自動的に保存され、いつでも続きから再開できます。

## 📂 プロジェクト構成と役割

開発の透明性とメンテナンス性を高めるため、以下のディレクトリ構造で整理されています。

```text
src/
├── components/
│   ├── map/
│   │   ├── WorldMap.tsx    # 地図表示・GPS追跡・ランク計算のメインロジック
│   │   └── FogOverlay.tsx  # Canvasを使用した「霧」の高速描画レイヤー
│   └── ui/
│       └── WelcomeModal.tsx # 初回起動時のチュートリアル・安全上の注意表示
├── styles/
│   ├── global.css          # 全体のレイアウト設定（フルスクリーン地図用）
│   └── App.css             # アプリケーション固有のスタイル
├── App.tsx                 # アプリのエントリーポイント
└── main.tsx                # Reactのレンダリング開始点
```

### 各ファイルの役割
- **WorldMap.tsx**: 
  Leafletを使用した地図表示、GPSによる現在地追跡、訪問ポイントの管理、探索度に基づいた「称号」の算出、昼夜モードの切り替えなど、アプリの核心となる機能が詰まっています。
- **FogOverlay.tsx**: 
  HTML5 Canvasを活用し、地図の動きに同期して「未踏の地の暗闇」と「訪れた場所の穴あき（霧消）」を高速に描画します。
- **WelcomeModal.tsx**: 
  新規ユーザーへの歓迎メッセージと、安全に使用するためのガイドラインを表示します。一度閉じるとブラウザに記憶（localStorage）されます。

## 🛡️ プライバシーと安全へのこだわり

- **完全プライバシー保護**: あなたの行動履歴（GPSデータ）は、**あなたのブラウザ内にのみ保存**されます。外部サーバーに送信されることは一切ないため、誰かに居場所を知られる心配はありません。
- **完全無料**: 地図データには OpenStreetMap を使用しており、全ての機能を無料でお使いいただけます。
- **安全第一**: チュートリアルに安全上の注意を記載。ユーザーの安全な冒険をサポートします。

## 🛠 使用技術（エッセンス）

- **Map Engine**: Leaflet (OpenStreetMap)
- **Visuals**: HTML5 Canvas API
